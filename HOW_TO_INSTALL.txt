================================================================================
  USAGE
================================================================================

HOW IT WORKS:
1. Extension blocks all websites except LeetCode
2. Solve problems on LeetCode
3. Extension automatically detects and counts them
4. Once you hit your target, all websites unlock!

TO SOLVE A PROBLEM:
1. Go to any LeetCode problem
2. Write your solution
3. Click "Submit" (NOT "Run Code")
4. Get "Accepted" status
5. Extension auto-counts it!

TO CHECK PROGRESS:
- Click extension icon
- Look at badge on icon (e.g., "0/1", "1/1")
- Red badge = not done
- Green badge = quota met!

TO TAKE A BREAK:
1. Click extension icon
2. Set break duration (1-240 minutes)
3. Click "Start Break"
4. Websites temporarily unlock
5. Click "End Break" to resume blocking

================================================================================
  TWO MODES EXPLAINED
================================================================================

ANY PROBLEM MODE:
- Solve ANY LeetCode problem to unlock
- Set custom targets (1-100 problems)
- Perfect for practice and learning

DAILY PROBLEM MODE:
- ONLY the official daily challenge counts
- Target is always 1 problem
- Other problems won't count
- Best for streak building

================================================================================
  TROUBLESHOOTING
================================================================================

PROBLEM: Extension failed to load
SOLUTION: Make sure you selected the folder with manifest.json

PROBLEM: Extension not blocking websites
SOLUTION: 
  1. Click extension icon
  2. Verify settings
  3. Click "Save Settings" again
  4. Refresh the page

PROBLEM: Problem not being detected
SOLUTION:
  1. Make sure you clicked "Submit" (not Run Code)
  2. Wait for "Accepted" status
  3. If Daily mode, check if it's the daily challenge

PROBLEM: Can't find extension icon
SOLUTION: Click puzzle icon (üß©) in toolbar and pin it
